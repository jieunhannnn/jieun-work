# 2026-06-29 Worklog

## Echo 캐릭터 이미지 생성봇

> 상세: [docs/character-echo/00-overview.md](../../../docs/character-echo/00-overview.md)

- Echo를 **자세·표정만 입력하면 일관되게 생성**하는 봇 제작 (마스터 프롬프트 + 베이스 이미지 내장).
- 베이스 이미지 3장 확보 → 그걸 토대로 마스터 프롬프트(캐릭터 바이블 v1.9) 정리.
- **1차**: HTML 생성봇 — 이미지 생성에 유료 API 필요해 실제 생성은 미검증.
- **1-1 임시방편(실패)**: 커스텀 GPT로 만들어봄 — 지식란 이미지를 시각 참조로 못 써서 다른 캐릭터가 나옴.
- **1-2 임시방편**: 별도 API 없이 쓰던 GPT 웹에 이미지 직접 첨부 + 요청 템플릿으로 수동 생성.
- **2차**: 브라우저 자동화(Image Starter, Playwright)로 "채팅 열기 + 이미지 첨부 + 프롬프트 입력"을 자동화 → 정상 동작 확인. 장면 한 줄만 입력하면 생성됨.
- 타 팀도 쓰도록 **빌드 명세서**(`tools/echo-image-starter/BUILD-WITH-AI.md`)로 재현 가능하게 정리.

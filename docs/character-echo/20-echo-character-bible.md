## 캐릭터 아이덴티티

Echo는 귀엽고 따뜻하며 친근한 토끼 마스코트다. 첫인상으로 **순수함 · 친근함 · 밝음 · 편안함 · 신뢰감**을 전달한다. 장난스럽더라도 과하게 익살스럽지 않으며, 항상 "같은 Echo"로 인식되어야 한다.

## 기본 디자인

- 둥글고 통통한 토끼, 2등신 SD 비율 (매우 큰 머리 / 매우 작은 몸 / 목 거의 없음)
- 둥근 형태 중심, 긴 둥근 토끼 귀 2개
- 민트색 브로콜리 모양 머리 장식 (시그니처)
- 연어빛 핑크 턱받이
- 눈: 크고 꽉 찬 다크 브라운 원형(흰자 거의 없음) + 작은 하이라이트 점, 작은 짙은 갈색 타원형 코
- 양 볼 블러셔, 손가락 항상 4개, 작은 둥근 꼬리
- Soft Clay 재질

## 실루엣

실루엣만 보아도 Echo여야 한다. 큰 원형 머리 / 작은 몸 / 짧은 팔·다리 / 둥근 귀·손·발. 모든 형태는 원을 조합한 듯 보이며, 직선적인 형태를 만들지 않는다.

### 비율 고정 (매 생성 동일하게 — 포즈보다 우선)

매 컷에서 **먼저 이 비율을 맞춘 뒤** 포즈·표정을 입힌다. 비율은 절대 변하지 않는 고정값이다.

- 전체는 항상 **2등신**: 머리 높이 ≈ 몸 높이의 약 1.4배 (머리가 확실히 더 큼)
- 머리 폭 : 몸통 폭 = **10 : 7** (몸이 머리보다 좁다)
- 귀 길이 = 머리 높이의 **70%** 고정 (이보다 길거나 짧게 금지)
- **목 길이 = 0** — 머리가 몸에 바로 붙는다
- 팔·다리는 몸통과 자연스럽게 이어진 짧고 둥근 형태, 길이 늘리지 않음
- 포즈가 바뀌어 몸이 약간 눌리거나 늘어나도 위 비율 관계는 유지한다

## 입 디자인 (v1.1)

입은 Echo의 감정을 표현하는 핵심 요소로, 항상 **One-piece** 디자인이다. 입술 없음, 윗/아랫입술 구분 없음 — 하나의 매끈한 곡선.

- **기본/차분:** 닫힌 부드러운 미소 곡선(살짝 올라간 U). 정면 기본 포즈가 이 형태
- **열린 입(신나거나 웃을 때만):** 입 안 진한 레드 / 혀는 밝은 코랄 핑크(아래쪽) / 치아·목구멍 없음
- 넓은 U자, 둥근 곡선, 양 끝이 자연스럽게 올라감, 아이콘처럼 단순
- **가능한 형태:** 기본 미소, 작은 열린 입, 활짝 웃음, 말하는 입, 놀란 O형, 감탄, 신난 웃음 — 모두 동일한 디자인 언어 유지

## 감정 표현용 요소 (Expressive Features)

Echo의 기본 얼굴은 단순·미니멀하다. 아래 요소들은 **강한 감정을 강조할 때만** 등장하는 **임시 표정 장치**이며, 영구적인 얼굴 요소가 아니다. 사용 후에는 기본 얼굴로 돌아온다.

### 눈썹
평소엔 보이지 않으며, 화남·결연·짜증·흥분·깊은 집중 등 강한 감정에서만 등장.
- 짧고 두툼한 둥근 캡슐 형태, 양 끝이 부드럽게 둥글다 (가늘거나 뾰족하거나 털·사실적 느낌 금지).
- 코·입과 동일한 다크 브라운.
- 눈 바로 위에 가깝게, 길이는 눈 하나 정도 폭.
- 감정에 따라 **각도만** 변함 — 두께·길이·색·스타일은 항상 동일.

### 불꽃 눈 (Flame Eyes)
열정·결의·흥분·승부욕이 강한 순간에 **눈동자를 불꽃 모양**으로 일시 표현.
- 기존 눈 안에 적용, **눈 외형 자체는 그대로** 유지.
- 단순한 둥근 불꽃 아이콘, 주황→노랑 그라데이션.
- 납작하고 깔끔한 마스코트풍 그래픽 — 사실적 불·발광 효과 금지.

### 찡그린 눈 ( >< )
힘주거나 안간힘 쓰는 과장된 표정에서 기본 둥근 눈을 **`><` 모양**으로 일시 대체.
- 두툼하고 둥근 선, 코·입과 동일한 다크 브라운.
- 좌우 대칭, 짧은 표현 순간에만 사용.
- 표정이 끝나면 기본 둥근 눈으로 복귀.

### 눈물
감동받거나 울컥하거나 우는 장면에서 사용.
- 작고 둥근 물방울 아이콘, **파스텔 하늘색 솔리드**(투명하지 않음).
- 작은 흰색 하이라이트 점 1개, 무광 클레이 질감.
- 볼 옆에 **붙어 있는 형태** (흘러내리지 않음).
- 투명·굴절·유리 효과, 사실적으로 흐르는 눈물 금지.

### 원칙
표정 요소는 감정을 더할 뿐, **Echo의 정체성을 바꾸지 않는다.** 얼굴 구조·비율·전체 인상은 모든 표정에서 동일하게 유지하며, 너무 날카롭거나 사나운 인상이 되지 않게 한다.

| 작은 열린 입 | ≈ 30 | ≈ 8 |
| 활짝 웃음 | ≈ 40 | ≈ 15 |
| 놀란 O형 | 원형 지름 ≈ 12 | — |

**그 외**

| 요소 | 수치 |
| --- | --- |
| 블러셔 지름 | ≈ 18 (눈 아래·코 높이 양옆, 위치·크기 고정) |

수치는 목표값이며 약간의 오차는 허용하되, **표정이 바뀌어도 이 비율 관계를 기준으로** 그린다.

## 귀 / 머리 장식 / 턱받이

- **귀:** 긴 캡슐 형태, 끝이 뾰족하지 않음, 좌우 동일 길이, 과하게 벌어지지 않고 살짝 안쪽
- **머리 장식:** 브로콜리처럼 보이는 둥근 민트색 (머리 폭의 약 45%). 잎사귀·머리카락·꽃·왕관으로 변형 금지
- **턱받이:** 연어빛 핑크, 꽃잎 3개처럼 생긴 둥근 형태. 절대 제거하지 않음

## 팔·손 / 다리·발 / 꼬리

- **팔·손:** 팔 짧음, 팔꿈치·손목 표현 없음, 손가락 항상 4개
- **다리·발:** 다리 짧음, 무릎 표현 없음, 발은 둥글고 통통
- **꼬리:** 작은 둥근 솜방울. 색은 **몸(Body)과 완전히 같은 Peach Pink `#F2AE9A` 단색** — 몸통에서 꼬리로 갈 때 색이 밝아지거나 하얘지지 않는다. **일반 토끼의 흰 솜꼬리가 아니다.** 흰색·아이보리·연한 회백색 모두 금지, 별도 하이라이트나 밝은 끝부분도 만들지 않는다. 조명 때문에 살짝 밝아 보이는 정도도 피하고, **몸통 색과 구분이 안 될 만큼 동일하게** 칠한다. 후면·측면·후면 3/4에서는 보이고, 정면·정면 3/4에서는 보이지 않음

## 재질 / 라이팅

- **재질:** Soft Matte Clay (부드러운 실리콘 질감). 플라스틱 / 털 표현 / 브러시 질감 금지
- **라이팅:** Soft Studio Lighting, Diffuse Shadow, Soft GI, Ambient Occlusion, Subsurface Scattering. 부드러운 그림자, 과도한 Rim Light 금지

## 소품 / 오브젝트 (Supporting Objects & Props)

Echo와 함께 등장하는 모든 소품·오브젝트는 **Echo와 같은 비주얼 세계관**에 속해야 한다. 일반 클립아트나 사실적인 물건이 아니라, **Echo를 만든 같은 디자이너가 만든 것처럼** 보여야 한다.

**핵심 원칙:** 어떤 소품이든 **사진처럼 사실적으로 렌더링하지 않는다.** 유리잔·음료·음식 같은 사물도 그릴 수 있지만, 실물을 그대로 옮기지 말고 **Echo와 같은 무광 클레이 마스코트 스타일로 단순화**해서 그린다. (예: 맥주잔 = 사실적 투명 유리 + 물방울 + 거품 ❌ → 매끈한 클레이 질감의 단순화된 잔 ✅)

- Echo와 동일한 **Soft Matte Clay 재질**.
- 동일하게 부드럽고 둥근 지오메트리, **날카로운 모서리 없음**.
- 단순하고 아이콘 같은 실루엣, 디테일 최소화.
- 살짝 크고 귀여운 비율.
- Echo와 동일한 라이팅·렌더링 스타일, 색은 부드럽고 약간 채도를 낮춤.
- 유리·액체 같은 소재도 **사실적 투명 유리·사진 같은 반사로 그리지 않는다** — 매끈한 클레이 질감으로 단순화. (은은하게 비치는 정도는 OK, 사진 같은 사실적 표현은 ❌)
- **물방울·응결·거품 디테일·사진 같은 반사·하이라이트 금지.**
- 사실적 텍스처·광택 플라스틱·금속·사진 같은 재질 금지.

## 효과 그래픽 (Effect Graphics)

감정·동작 표현 효과(별, 불꽃, 느낌표, 땀방울, 모션 라인, 반짝임 등)도 Echo의 디자인 언어를 따른다.

- 둥근 형태, 부드러운 마스코트 비율.
- 미니멀 아이콘 디자인, 플랫하거나 소프트 클레이 느낌.
- 만화책(코믹) 스타일 효과 금지, 사실적 VFX 금지.

## 표정 자동 매칭

장면에 **행동만 주어지고 표정이 지정되지 않으면**, 행동·상황에 가장 어울리는 표정을 자동으로 골라 적용한다. (표정이 명시되면 그대로 따르며, 자동 매칭보다 우선한다.)

- 달리기·점프·춤 → 신나는 웃음 (열린 입)
- 박수·브이·금메달 → 활짝 웃음, 들뜬 표정
- 무거운 것 들기·안간힘 → 찡그린 눈(`><`), 힘주는 입
- 졸기·눕기 → 나른한 눈, 편안한 미소
- 감동·울컥 → 글썽이는 눈물 + 미소
- 고민·생각 → 살짝 갸웃, 다문 입
- 화남·결연 → 눈썹 등장(각도만), 단단한 입

어떤 표정이든 **기본 인상(순수·둥글·친근)은 유지** — 사납거나 과한 표정은 피한다.

## 배경 (기본: 없음)

**배경은 요청하지 않으면 그리지 않는다.** 기본값은 **깔끔한 단색/흰색(또는 투명) 배경**이며, 캐릭터만 단독으로 배치한다.

- **행동이 특정 환경을 연상시켜도(서핑·수영·등산·카페 등) 바다·파도·풍경·실내 같은 배경을 임의로 추가하지 않는다.** 서핑이면 서핑보드까지만, 배경 바다는 그리지 않는다.
- 배경은 오직 **변형 지시에서 명시적으로 요청했을 때만** 그린다.
- 배경을 그릴 때도 사실적 풍경이 아니라, Echo와 같은 **단순한 무광 클레이/파스텔 톤**으로.

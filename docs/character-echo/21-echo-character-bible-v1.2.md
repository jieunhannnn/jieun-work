# Echo Character Bible v1.2

> 이미지 생성용 마스터 프롬프트. 제공된 참고 이미지를 **절대 기준(Canonical Reference / Model Sheet)** 으로 사용한다.
> 목표는 새 토끼를 디자인하는 것이 아니라, **기존 Echo가 다양한 상황에서 연기하는 모습**을 만드는 것이다.
> 포즈·행동·표정·카메라 시점은 자유롭게 바꾸되, 캐릭터 아이덴티티는 항상 유지한다.
> 참고 이미지는 검정 배경일 수 있으나, 생성 결과의 기본 배경은 **흰 배경 또는 투명 배경**으로 둔다.

## 캐릭터 아이덴티티

Echo는 귀엽고 따뜻하며 친근한 토끼 마스코트다. 첫인상으로 **순수함 · 친근함 · 밝음 · 편안함 · 신뢰감**을 전달한다. 장난스럽더라도 과하게 익살스럽지 않으며, 항상 "같은 Echo"로 인식되어야 한다.

## 기본 디자인

- 둥글고 통통한 토끼, 2등신 SD 비율 (매우 큰 머리 / 매우 작은 몸 / 목 거의 없음)
- 둥근 형태 중심, 긴 둥근 토끼 귀 2개. 귀 안쪽 패치나 선은 없음
- 민트색 3-lobed 브로콜리 캡 모양 머리 장식 (시그니처)
- 연어빛 핑크 턱받이
- 큰 흰 눈 + 작은 짙은 갈색 동공, 작은 짙은 갈색 타원형 코
- 양 볼 블러셔, 둥근 벙어리장갑 같은 손, 작은 둥근 꼬리
- Soft Clay 재질

## 실루엣

실루엣만 보아도 Echo여야 한다. 큰 원형 머리 / 작은 몸 / 짧은 팔·다리 / 둥근 귀·손·발. 모든 형태는 원을 조합한 듯 보이며, 직선적인 형태를 만들지 않는다.

## 얼굴 구조

| 기준(얼굴 폭=100) | 위치 |
| --- | --- |
| 눈 | 약 35 |
| 코 | 약 49 |
| 입 | 약 60 |

- 눈: 큰 흰 타원 + 작은 세로 타원 동공
- 눈은 얼굴 중앙에 모여 있고, 눈 사이 간격은 눈 하나 정도
- 눈 위치를 크게 변경하지 않는다
- 속눈썹, 눈썹, 반짝이는 애니메이션식 하이라이트를 추가하지 않는다

## 입 디자인 (v1.2)

입은 Echo의 감정을 표현하는 핵심 요소다. 기본 입은 **짙은 갈색의 단순한 U자형 미소**이며, 얼굴 위에 얹힌 매끈한 클레이 조각처럼 보인다. 입술 없음, 윗/아랫입술 구분 없음, 치아 없음, 혀 없음, 목구멍 없음.

- **기본 형태:** 두툼한 U자, 둥근 곡선, 양 끝이 자연스럽게 올라감, 아이콘처럼 단순
- **기본 내부:** 입 안 색, 혀, 치아, 목구멍 디테일 없음
- **가능한 변형:** 작은 열린 입, 활짝 웃음, 말하는 입, 놀란 O형, 감탄, 신난 웃음
- **변형 규칙:** 열린 입이 필요할 때도 하나의 단순한 덩어리 형태를 유지한다. 입 안은 과하게 깊게 만들지 않고, 치아·목구멍·입술·과도한 혀 디테일은 만들지 않는다.

## 귀 / 머리 장식 / 턱받이

- **귀:** 긴 캡슐 형태, 끝이 뾰족하지 않음, 좌우 동일 길이, 과하게 벌어지지 않고 살짝 안쪽. Body와 같은 Peach Pink이며 귀 안쪽 색 패치나 라인을 추가하지 않음
- **머리 장식:** 브로콜리처럼 보이는 매끈한 3-lobed 민트색 클레이 캡 (머리 폭의 약 45%). 잎사귀·머리카락·꽃·왕관·채소 질감으로 변형 금지
- **턱받이:** 연어빛 핑크, 꽃잎 3개처럼 생긴 둥근 형태. 절대 제거하지 않음

## 팔·손 / 다리·발 / 꼬리

- **팔·손:** 팔 짧음, 팔꿈치·손목 표현 없음. 손은 둥근 벙어리장갑처럼 보이며, 필요할 때만 4개의 짧고 둥근 손가락을 암시한다. 길고 분리된 사람 손가락 금지
- **다리·발:** 다리 짧음, 무릎 표현 없음, 발은 둥글고 통통. 다리는 몸통과 자연스럽게 이어진 짧은 둥근 기둥처럼 보임
- **꼬리:** 작은 둥근 솜방울, Body와 같은 Peach Pink. 조명 때문에 약간 밝게 보일 수 있지만 흰 꼬리로 만들지 않는다. 후면·측면·후면 3/4에서는 보이고, 정면·정면 3/4에서는 보이지 않음

## 재질 / 라이팅

- **재질:** Soft Matte Clay (부드러운 실리콘 질감). 플라스틱 / 털 표현 / 브러시 질감 금지
- **라이팅:** Soft Studio Lighting, Diffuse Shadow, Soft GI, Ambient Occlusion, Subsurface Scattering. 부드러운 그림자, 과도한 Rim Light 금지

## 표정 / 포즈 / 시점

- **표정:** 😊 😄 🥹 🤔 😮 😍 😴 🥳 😢 🤩 — 표정이 변해도 눈 위치·입 디자인·얼굴 비율·캐릭터 인상은 유지
- **포즈:** 걷기 / 달리기 / 점프 / 춤 / 브이 / 엄지척 / 박수 / 손 흔들기 / 앉기 / 눕기 / 스트레칭 — 몸 약간 압축 가능하나 기본 비율 유지
- **시점:** 정면 / 3/4 / 측면 / 후면 / 로우앵글 / 하이앵글 — 모든 시점에서 동일한 Echo로 인식

## 가변 템플릿

위 아이덴티티를 유지한 채, 아래 항목만 바꿔 장면을 생성한다.

```
행동:   [걷기 / 뛰기 / 손 흔들기 / 박수 / 점프 / 앉기 ...]
표정:   [기본 / 웃음 / 놀람 / 감동 / 고민 / 졸림 ...]
시점:   [정면 / 3/4 / 측면 / 후면 / 하이앵글 / 로우앵글]
분위기: [밝게 / 차분하게 / 활기차게 / 귀엽게 ...]
배경:   [흰 배경 / 투명 / 단색 / 간단한 소품]
출력 목적: [앱 UI / 스티커 / 광고 / 일러스트 / 아이콘]
```

### 생성 원칙

- 새로운 토끼를 만들지 않는다.
- 기존 Echo가 연기하는 장면을 만든다.
- 캐릭터를 재해석하지 않는다.
- 참고 이미지를 Canonical Model Sheet로 사용한다.
- 모든 이미지에서 동일한 Echo임을 즉시 알아볼 수 있어야 한다.

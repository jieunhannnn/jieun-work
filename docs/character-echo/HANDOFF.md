## 알려진 한계 (프롬프트로 100% 못 잡는 것)

- **흰 꼬리** — "토끼=흰꼬리" 학습이 강해 규칙 강화해도 가끔 샘. 사용자는 "흐린 눈"으로 넘어가기로 함.
- **손-소품 잡기** — 맥주잔 등이 손에 안 잡히고 뜸. 손 동작은 모델 약점, 재생성이 답.
- **표정 다양성** — 베이스 3장이 웃는 계열이라 슬픔·분노가 크게 안 벌어짐. 근본 해결은 "표정 베이스 추가"지만, 사용자는 지금은 안 하기로 함.
